# CODSOFT Artificial Intelligence Internship

## Overview

This repository contains the Artificial Intelligence projects I completed as part of my **CODSOFT AI Internship**. These projects demonstrate practical applications of Artificial Intelligence, Machine Learning, Computer Vision, and Natural Language Processing using Python.

## Completed Tasks

### Task 3 – Image Captioning AI

**Description**

Developed an AI-based image captioning system that automatically generates descriptive captions for input images by combining Computer Vision and Natural Language Processing.

**Key Features**

* Generates captions for images.
* Uses a pre-trained BLIP model.
* Accepts JPG, JPEG, and PNG image formats.

**Technologies Used**

* Python
* Transformers (Hugging Face)
* PyTorch
* Pillow

---

### Task 4 – Movie Recommendation System

**Description**

Built a content-based recommendation system that suggests similar movies based on their genres using machine learning techniques.

**Key Features**

* Recommends similar movies.
* Uses cosine similarity for recommendations.
* Simple command-line interface.

**Technologies Used**

* Python
* Pandas
* Scikit-learn

---

### Task 5 – Face Recognition System

**Description**

Developed a real-time face recognition system using OpenCV. The application captures face samples through a webcam, trains an LBPH face recognizer, and identifies registered users.

**Key Features**

* Face dataset collection.
* Model training.
* Real-time face recognition using a webcam.

**Technologies Used**

* Python
* OpenCV
* NumPy
